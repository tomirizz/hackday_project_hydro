# Цифровой каталог гидротехнических сооружений

Система мониторинга и учёта технического состояния гидротехнических сооружений
и гидропостов Жамбылской области для РГП «Казводхоз».

## Структура

```
backend/
  schema.sql          схема БД
  parse_dataset.py    парсинг каналов из датасета Казводхоз
  parse_hydroposts.py парсинг гидропостов (отдельный датасет)
  main.py             FastAPI сервер
  models.py           ML-модели (risk, inspection period)
frontend/
  src/                React + Leaflet
data/
  objects.csv         438 каналов (генерируется parse_dataset.py)
  database.db         SQLite (генерируется при запуске бэкенда)
```

## Быстрый старт

```bash
# 1. Парсинг датасета
python backend/parse_dataset.py

# 2. Бэкенд
pip install -r requirements.txt
export ANTHROPIC_API_KEY="ваш-ключ"   # для ГидроБота
uvicorn backend.main:app --reload

# 3. Фронтенд
cd frontend && npm install && npm start
```

Деплой на сервер не требуется — система запускается локально.
ГидроБот работает на Claude API (нужен ключ ANTHROPIC_API_KEY).
Без ключа всё остальное работает, только чат-ассистент недоступен.

## Стек

- **Backend:** Python, FastAPI, SQLite
- **Frontend:** React, Leaflet (OpenStreetMap)
- **ML:** pandas, scikit-learn
- **AI:** Claude API (ИИ-ассистент)
- **Reports:** ReportLab

## Данные

Два типа объектов:
- **Каналы** (438 объектов из таблицы Казводхоз)
- **Гидропосты** — измерительные станции с датчиками воды (отдельный датасет)

Состояние каналов: 334 удовл. (76%) / 100 неудовл. (23%)  
Годы постройки: 1900–2008  
КПД проектный → фактический: 0.71 → 0.58 (среднее падение 0.132)

Координаты в исходном датасете отсутствуют — генерируются
детерминированно в границах Жамбылской области (42.5–45°N, 69.5–74.5°E).
